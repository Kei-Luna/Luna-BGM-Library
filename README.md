# Luna-BGM-Library

It's basically just an audio player.  

It has the following features  
 - Unpack the PCK files in the PCK folder and output them in flac format to the BGM folder.  
 - Equalizer
 - Ability to download background music from various games (Not implemented yet)


## Requirements

This application requires the Microsoft [.NET 8.0](https://dotnet.microsoft.com/download/dotnet/8.0).
